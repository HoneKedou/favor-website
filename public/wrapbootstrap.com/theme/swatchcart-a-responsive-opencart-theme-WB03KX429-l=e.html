SwatchCart - a responsive Bootstrap 3 OpenCart theme

* SwatchCart
* Version: 1.0 from 27th January 2014
* Author: appman
* Website: ExpressPixel.com

The combination of Bootstrap 3 and OpenCart 1.5+ allows you to build your own eCommerce site that's compatible on all devices with ease.

DESCRIPTION

SwatchCart is an OpenCart theme designed entirely using Bootstrap 3.0.3 with minimal customizations. It's fully responsive and works on all devices. It's compatible with any of the themes from http://bootswatch.com/ making it easy for you to change the theme. You can also easily add and remove components using OpenCart.

Key features

  • Fully responsive eCommerce shopping cart
  • Minimalist design style, loading fast
  • Compatible with any Bootswatch theme
  • Falls back onto existing OpenCart styles for custom extensions.
  • Uses LESS for even easier customization

Pages

All OpenCart pages are complete. In the case you add new extensions it'll use the original OpenCart stylesheets

  • Product details
  • Shopping cart
  • Product category
  • Product listings
  • Checkout
  • Product comparison
  • Contact us
  • Login and registration
  • My Account
  • Affiliation

Code

  • Compatible with Bootstrap 3.0.x
  • Uses Bootstrap 3.0.3
  • Written in valid HTML5 that makes good use of proper semantics
  • Theme modifications are stored in a separate CSS stylesheet to make upgrading easy
  • View the layout without styles to see the well-formed, bare markup

Browsers

  • Firefox
  • Latest Chrome
  • Latest Safari
  • Internet Explorer 8+
  • Opera 11

Assets

  • OpenCart - http://www.opencart.com/
  • Bootstrap Twitter - http://twitter.github.com/bootstrap/
  • Bootswatch - http://bootswatch.com/
  • jQuery - http://jquery.com/

Installation

  • Upload this theme folder 'swatchcart' to the opencart/catalog/view/theme/ folder and activate the theme.
  • Please read the docs for more information : http://docs.expresspixel.com/swatchcart

Note: No product images are included with the theme. They are included with OpenCart's own sample content.

Don't forget! Because you're using a theme built with Twitter Bootstrap, you get access to everything in the Bootstrap component library and ecosystem. The use of OpenCart makes it very easy to get started with your own web store in very little time. The combination makes SwatchCart an excellent choice for your next project. It's simple to customize and change themes.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
