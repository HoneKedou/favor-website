OlsonAdmin Features:

  1. Minimal Admin theme. Other Olson Themes - Olson / Olson Kart
  2. Great typography.
  3. Fast loading theme ever.
  4. Bar chart generated using jQuery flot plugin.
  5. It also support pie chart, real time chart, curve chart.
  6. All charts are fully responsive.
  7. jQuery peity plugin for mini charts.
  8. Simple sidebar menu with dropdown.
  9. Base file included to create your own pages.
  10. Simple 404 error page.
  11. Calendar page with Google calendar using Full Calendar plugin.
  12. Error log page to list out errors generated with Color highlights.
  13. Simple form page which list out all form fields.
  14. This theme supports CL editor for form.
  15. Simple gallery page to showcase the images.
  16. Invoice page to create and send invoice.
  17. Simple login page.
  18. Make post page to create post.
  19. Make post page supports fields for excerpt, custom fields, category, tags, etc.,
  20. Media page to list out all media files in table.
  21. Profile page to view profile and too form for editing it.
  22. Registration page for new users to get registered.
  23. Table page showcase different table styles.
  24. Widget page lists out all widgets.
  25. Chat widget to chat with your members.
  26. Quick post widget to make posts on the fly.
  27. Project widget list the projects with progress bar indicating completion.
  28. Server widget about server status.
  29. Browser widget shows users browser.
  30. Recent activity widget list out the activities of the users.
  31. Map widget with google map.
  32. Operating system widget shows users OS.
  33. Recent posts widget.
  34. Task widget with list of tasks.
  35. Toggle button uses bootstrap switch plugin.
  36. Horizontal slider and vertical slider using jQuery UI plugin.
  37. Progress bar and incrementing progress bar.
  38. Star rating plugin.
  39. Date picker and timer picker.
  40. Gritter plugin for different types of notification.
  41. Uses google web fonts.
  42. Uses font awesome icon.
  43. Fully based on bootstrap.
  44. Fully responsive.

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
