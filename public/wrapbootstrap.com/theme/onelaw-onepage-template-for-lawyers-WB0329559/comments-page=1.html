Be prepared to use this theme almost exactly as it's shown in the preview, because the customization is limited to: (1) changing colors of the buttons; (2) changing the layout from full screen to box screen with changeable backgrounds; (3) changing the text content; (4) inserting pictures (apparently without capability for resizing); and (5) deleting sections of the site- at your own risk and peril of ruining the sites functionality or layout.

If you have a general familiarity with html and css, also be prepared to not know what you're doing at times. While this theme is touted as easy to customize and laid out in simple html- the reality is that it's layered in multiple css and javascript files that are not described in the guide and are difficult to figure out for laypeople. Even simple things like changing text color or font size require a great deal of work.

Good luck trying to adapt this template to your needs- it's filled with bugs (e.g. the contact form doesn't include the CAPTCHA code so it does not work); it's inaccessible; and the provided guide is basically useless. Don't expect help from the developers either- they'll likely refer you to the guide or ignore your problems. 2/5

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
