A Robust Multi-Purpose Business Template

reDesign is a robust multi-purpose business template ready to help your site compete in the competitive web. It's filled with 27 page templates built with HTML5 and CSS3 coding using the Twitter Bootstrap Framework. The design is perfect for creating a business site with bonus product and service pages totally unique to reDesign.

We include professional documentation to help you create a new site with ease. reDesign is all you need to help your business blast off.

27 Valid Page Templates Including 6 Incredible and Exclusive Bonus Pages for Promoting Products and Services

reDesign Includes beautifully crafted page templates, including 6 highly unique page templates specifically designed to help you better promote and manage your businesses' products and/or services...

Standard Pages

  • Home – the main/primary home page template.
  • Home Alternate Page 1 – an alternative home page template (index2.html).
  • Home Alternate Page 2 – an alternative home page template (index3.html).
  • Home Footer 1 Page – a home page template with an alternative footer background (index-footer1.html)
  • Home Footer 2 Page – a home page template with an alternative footer background (index-footer1.html).
  • Home Footer 3 Page – a home page template with an alternative footer background (index-footer1.html)
  • Home Footer 4 Page – a home page template with an alternative footer background (index-footer1.html).
  • Shortcodes Page – a page template that breaks down several popular Twitter Bootstrap shortcode elements which can used within in reDesign web page you create (shortcodes.html).
  • Masonry-style Portfolio Page - a masonry-style portfolio template (portfolio-masonry.html).
  • Grid-style Portfolio Page – a 4-column grid portfolio grid template (portfolio-grid.html).
  • Portfolio Item Page – a page template for displaying a single portfolio item (portfolio-item.html).
  • About Us Page – an “about us” or “company” page template (about.html).
  • Left Sidebar Page – a basic page template with left-column sidebar (left-sidebar.html).
  • Right Sidebar Page - a basic page template with right-column sidebar (right-sidebar.html).
  • Both Sidebars Page - a basic page template with both left and right-column sidebars (both-sidebars.html).
  • Team Row Page – a page template designed to list your company’s key team members (team-1-column-full-page.html).
  • Team 2-Column Page – an alternate page template designed to list your company’s key team members (team-3.html).
  • Team 2-Column Centered Page - a page template designed to list your company’s key team members (team-3-centered.html).
  • Team Row with Left Sidebar Page - a page template designed to list your company’s key team members (team-2.html).
  • Gallery Page – a basic image gallery page template with lightbox controls (gallery.html).
  • Contact Us Page – a “contact us” page template, form and display map (contactus.html).

Bonus Pages

  • Feature Tour Page – a page template that allows you to create a guided tour (product-tour.html).
  • Product History Page – a page template that allows you to create a historical timeline, which can be used for product or company history (product-release-history.html).
  • Feature Comparison Page – a page template that allows you to create a product or service comparison chart (product-comparison.html).
  • Pricing Tables Page – a page template that allows you to create a product and/or service pricing table (pricing-tables.html).
  • FAQs/Support Page – a page template for that allows you to create a portal for support and/or product and service FAQs (faqs-support.html).

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
