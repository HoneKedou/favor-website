In Swedish the word Enkel means "Simple" so that's exactly what this theme is, simple but also elegant, powerful & easy to customise.

Enkel comes with 5 different colour options (blue which is default, red, grey, purple & green) and starter skeleton CSS and HTML files are provided to get you started with customisation.

Code is well commented and readme files are provided within the theme bundle.

The theme is ideal for a company website but its simple design & ease with which it can customised make it a great choice for any type of site.

Key Features

  • Built on Bootstrap 2.3.1
  • Font Awesome icons
  • Bootstrap JavaScript implemented: tooltip, popovers, carousel
  • Fully responsive (wide, normal, narrow, mobile)
  • Flexslider Slideshow
  • 5 different colour schemes
  • Easy to customise
  • Start template (starter.htm)
  • jQuery Quicksand plugin
  • Simple & clean design
  • Valid HTML 5
  • Google Font

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
