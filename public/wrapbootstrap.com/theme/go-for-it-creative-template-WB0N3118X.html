Go For It - Creative Template

Extended license only $99

Go For It is a custom made creative template based on bootstrap 2.3 that has some very cool features to offer. It`s a one page site template with sticky navigation bar

and big fixed background image in the homepage. Uses jquery to scroll smoothly to individual sections of the site. Clean and simple design.

POWERFUL FEATURES

The most powerful feature is custom modified codrops plugin that looks like google images search and offers these functions:

  • HOVER EFFECT WITH CUSTOM TEXT FOR BIG THUMBNAILS
  • EXPANDING PREVIEW WITH MORE SMALL THUMBNAILS IN THE ONE OPENED PROJECT - LIKE GALLERY
  • CHANGE TEXT TO EACH THUMBNAILS DETAIL
  • TURN ON AND OFF THE LINK BUTTON
  • BROWSE THE PORTFOLIO WITH NEXT AND PREV ARROWS
  • SUPER EASY HTML STRUCTURE TO ADD MORE THUMBS TO EACH PROJECT

Write most important information or quotes to the blocks with bigger text and custom image background.

Write about what`s going on with a nice timeline plugin.

Whole site is completely responsive for all the devices and screen sizes .

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
