LandingPad - Landing Page

  • Details
  • Comments New
Live preview  Purchase now »

LandingPad is a one-page introduction to your service to get people on board. Designed to be straight-forward, easy to edit via LESS, and fast to set-up for immediate use. It works instantly out of the box. Simply un-zip the package then put in your copy and screenshots.

Contains

  • 'Open Sans' font from FontSquirrel.
  • Icon Fonts from Icomoon
  • Main Landing page: index.html
  • Simple Form pages: login.html, forgotpassword.html, signup.html
  • Twitter Bootstrap 2.3.1, jQuery 1.9.1 (minified) and jQuery gMaps 2.1.2 (minified)
  • Important Note: does not contain iPhone and Chrome browser PSD templates.

Features

  • Responsive layout and tested across web browsers.
  • CSS is compiled with LESS for easier editing of colors and styles. login.less has all the basic styles you'll need for creating simple form pages, pad.less contains the styles for the main landing page.
  • Uses Twitter Bootstrap's Carousel javascript component for a section that may contain your screenshots, use cases or testominals.
  • Plug-and-play: LandingPad's CSS is in a different file, allowing you to plug it in and out with other themes.
  • HTML5: Classes can be used with HTML5 tags (header, section and article)

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
