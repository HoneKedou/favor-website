Feautres

  • Four color styles, something to suit any mood.

  • Fully responsive.
  • Google webfont PT Sans.
  • Browser component for displaying screenshots done in pure CSS. Stand out from the sea of Mac's and iMac's used for this purpose.
  • Custom icons and imagery designed specifically for Colorfrog, clean and easy to understand. Adoble Illustrator (.ai) resource package included.
  • Documentation included for customising theme parts to suit your needs best.

Templates

Basic:

The required templates of most websites:

  • Home page
  • About
  • Contact
  • Sub page

Blog

A complete set of templates allowing you to get a blog up and running, including a blog main page, post single page and three archive templates:

  • Blog
  • Blog post
  • Blog archive (by year)
  • Blog archive (by month)
  • Blog archive (by author)

Clients / Portfolio

Templates suitable for showcasing either client work or portfolio items. There is a main page where you can list everything followed by a single view template allowing for greater detail:

  • Clients / portfolio
  • Single client / portfolio item

Products

Templates suitable for showcasing your cutting edge products/services. A main page allows you to give an overview of each and a single product/service page allows you to go into greater detail:

  • Products / services
  • Product / service single

Pricing

Layout the benefits for your clients in one of two pricing package templates. First, a typical three package layout followed by a tabular comparison:

  • Pricing - 3 packages
  • Pricing - table comparison

Support

Good customer service is often the cornerstone of any business, we give you a series of templates to help you out. A main page groups together the main support areas followed by a dedicated FAQ template and documentation template:

  • Support
  • FAQ
  • Documentation

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
