Cotton WordPress

digitalcookers™ finally presents their first WordPress theme coming out of the oven. Better late than never, right?

Cotton for WordPress was designed and built to be easy to use and customize. We did our best to provide options to tweak the theme as much as possible for every kind of potential buyer. For us nothing is a complete product so we look forward for your feedback and we will update the theme with extra features as Cotton WordPress goes down the road.

Like the HTML it's minimal, elegant, professional, flat designed, HTML5 / CSS3 single page template, fully adaptable to all major mobile devices. Designed for creatives, digital agencies, photography, illustration or other type of business to present themselves and their work.

3 months to cook - we almost got it burn! Enjoy.

P.S. If you like it, please don't forget to comment, recommend and follow our shop! We'll truly appreciate it! digitalcookers™

WORDPRESS FEATURES:

  • VIDEO DOCUMENTATION - Setup your theme in 10min - http://youtu.be/16fBXrFjCr4
  • Moved from Foundation 4.0 to Boostrap 3.0
  • Easily change the settings with advanced theme options and individual section, post and page options
  • Choose any font from Google Fonts - list is always updated
  • Easily add custom styles in the theme options without having to get your hands on the css file
  • The same scrollTo Sections: team, services, studio, portfolio, social & contact and NOW you can choose which sections you want to have or simply mix the options to build different sections
  • Off-Canvas Navigation for Pages
  • Blog section implemented with an individual post page carefully designed for good reading
  • Blog section has mansonry to easily adjust to grid and also on resize for different heights
  • Post Formats Supported: Video, Gallery, Image, Audio and Standard
  • Post Format - Video: choose videos from Youtube/Vimeo or upload your own
  • Post Format - Audio: upload your own music or link to mp3 file
  • Post Format - Gallery: create a Swiper slideshow from a drag and drop gallery
  • FontAwesome implemented for easy playing with the icons http://fortawesome.github.io/Font-Awesome/icons/

OTHER FEATURES:

  • Mobile-First Design Aproach
  • Fully Responsive layout
  • Valid HTML5/CSS3
  • Allways-on-top Navigation
  • Swiper by iDangerous: Touch Enabled Slider
  • Sortable & Filtered Portfolio Section powred by MixItUp plugin
  • Dedicated work page with Ajax loading. Show not only a larger preview of your work, but a well documented description of it with any type of media that you like
  • Instagram, Twitter & Flickr Feed
  • Fully working PHP Contact Form with AJAX validation

About Bootstrap

Bootstrap is designed to help people of all skill levels – designer or developer, huge nerd or early beginner. Use it as a complete kit or use it to start something more complex.

Unlike other front-end toolkits, Bootstrap was designed first and foremost as a styleguide to document not only its features, but best practices and living, coded examples.

Built to support new HTML5 elements and syntax with progressively enhanced components.

Browser support

Bootstrap is tested and supported in major modern browsers like Chrome, Firefox, and Internet Explorer.

  • Latest Chrome
  • Latest Safari
  • Latest Firefox
  • Internet Explorer 8/9/10/11
  • Latest Opera

Features & includes

Bootstrap provides an unparalleled array of features and reusable components that let you hit the ground running:

  • Responsive 12-column grid
  • 12 custom, modular jQuery plugins
  • CSS styles for forms, navigation & more
  • Dozens of reusable components
  • Components are scaled according to a range of resolutions and devices
  • Built with LESS and Sass (CSS included)
  • Complete styleguide documentation
