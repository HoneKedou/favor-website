Description:

Aquantic is a logo template that matches fitness, sports, athletics, gym-studio, as well as water fitness, swimming-pool related activities and yoga.

It's intention is to represent integrated movement, dynamic actions & connections through exercises and movement

RECOMMENDATION:

Recommended is to use this logo for the following sectors:

  • Fitness Studio
  • Swimming, aquatic sports
  • Yoga
  • Athletics
  • Gym-studio
  • Swimming-pool center

File specifications:

The logo is made in 2 version:

  • Vertical approach over transparent background
  • Horizontal approach over transparent background.

FILE TYPES:

All files are vector data for .PSD, .EPS, .AI and .PNG (Adobe-Fireworks with pages)

FONTS:

The font used is "EXO 2" in Black style (Google fonts)

COLORS:

Colors used are:

  • Teal-green: #00cfdc;
  • Azure-Violet: #6854ec;
