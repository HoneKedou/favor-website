Description:

Cornerstone is a logo template that can match all styles needed, as it is a generalized logo proposal.

File specifications:

The logo is made in 3 version:

  • Vertical approach over transparent background
  • Horizontal approach over transparent background
  • Variations with B&W, resizing options down to favicon size (16x16), as well as colored options

FILE TYPES:

All files are vector data for .PSD, .EPS, .AI, .PDF and .PNG (Adobe-Fireworks with pages)

FONTS:

The font used is "Raleway" in Heavy and Regular style (Google fonts)

COLORS:

Main (Blue) Colors used are:

  • Bright Blue: #51b7ff;
  • Medium Blue: #167cc4;
  • Darker Blue: #002946;
  • And 4 additional color options with 3 tone variations on each one.
